query :Object

get params pass by querystring, all values are of string type.

Source:
  • app/extend/request.js, line 194
Example
```js
GET http://127.0.0.1:7001?name=Foo&age=20&age=21
this.query
=> { 'name': 'Foo', 'age': '20' }

GET http://127.0.0.1:7001?a=b&a=c&o[foo]=bar&b[]=1&b[]=2&e=val
this.query
=>
{
  "a": "b",
  "o[foo]": "bar",
  "b[]": "1",
  "e": "val"
}
```
